Thoracle

Senior Design Project

Thoracle is a multi-platform application that enables users to interact with tweet screenshots. For a given tweet screenshot, Thoracle will provide the Twitter link if the corresponding tweet exists. On the other hand, it will inform the user if the tweet screenshot is fake, or if the corresponding tweet is deleted. Thoracle will have add-ons to help the user detect fake news in tweet contents. These add-ons include an AI-powered classifier and a validity discussion forum. One of the goals of Thoracle project is preventing public manipulation by fake news through social media. Besides, Thoracle will turn the non-interactive tweet screenshots into real tweets, which can be liked, shared and mentioned in a more convenient way.
